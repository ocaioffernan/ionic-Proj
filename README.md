# Funcionalidades
- F 1. Tirar Foto (Não armazena)
- F 2. Ligar/Desligar Flash
- F 3. Breve Biografia

# Instalação
```

1. Clone o repositório:

git clone https://github.com/ocaioffernan/ionic-Proj.git

2. Acesse o diretório do projeto:

cd ionic-Proj

3. Instale as dependências:

npm install

4. Execute o projeto:

ionic serve
